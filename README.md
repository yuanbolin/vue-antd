## 运行使用
### yarn
```bash
$ yarn install
$ yarn serve
```
### or npm
```
$ npm install
$ npm run serve
```

## 脚手架文档
### yarn
```bash
$ yarn docs:dev
浏览器访问 http://localhost:8080/vue-antd-docs/ 
```
### or npm
```
$ npm run docs:dev
浏览器访问 http://localhost:8080/vue-antd-docs/ 
```
